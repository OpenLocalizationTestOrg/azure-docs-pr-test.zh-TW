# <a name="azure-relay-port-settings"></a>Azure 轉送連接埠設定

下表說明 Azure 轉送的連接埠值所需的設定。

## <a name="hybrid-connections"></a>混合式連線
混合式連線使用 WebSockets 做為基礎傳輸機制，僅使用 **HTTPS**。
