## <a name="compute-management-and-job-scheduling"></a><span data-ttu-id="41140-169">計算管理和作業排程</span><span class="sxs-lookup"><span data-stu-id="41140-169">Compute management and job scheduling</span></span>
<span data-ttu-id="41140-170">執行 Batch 和 HPC 應用程式通常包括「叢集管理員」和「作業排程器」來協助管理叢集的計算資源，以及將其配置給執行作業的應用程式。</span><span class="sxs-lookup"><span data-stu-id="41140-170">Running Batch and HPC applications often includes a *cluster manager* and a *job scheduler* to help manage clustered compute resources and allocate them to the applications that run the jobs.</span></span> <span data-ttu-id="41140-171">這些函式可能會透過不同的工具或某種整合式工具或服務來完成。</span><span class="sxs-lookup"><span data-stu-id="41140-171">These functions might be accomplished by separate tools, or an integrated tool or service.</span></span>
