# <a name="what-is-azure-stack"></a>什麼是 Azure Stack？

Microsoft Azure Stack 是混合式雲端平台產品，可讓您組織的資料中心提供 Azure 服務。  Azure 堆疊是設計的 toohelp 您在索引鍵的情況下，像是符合安全性與相容性需求，或您需要 tooaccess 未連線到網際網路的 Azure 資源的位置。  

## <a name="azure-stack-development-kit"></a>Azure Stack 開發套件
Microsoft Azure 堆疊開發套件是單一節點版本的 Azure 堆疊，您可以使用 tooevaluate 並了解 Azure 堆疊。  您也可以將 Azure Stack 開發套件作為開發人員環境，您可在其中使用一致的 API 和工具進行開發。  

您應該留意下列 Azure Stack 開發套件事宜：

* Azure Stack 開發套件不可用於生產環境，而應僅用於測試、評估和示範。  
* Azure Stack 的部署與單一 Azure Active Directory 或 Active Directory 同盟服務識別提供者相關聯。 您可以在此目錄中建立多個使用者並指派訂閱 tooeach 使用者。
