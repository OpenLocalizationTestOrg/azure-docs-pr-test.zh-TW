# <a name="azure-remoteapp-retirement"></a>Azure RemoteApp 淘汰
您會被導向這篇文章的原因，是因為我們無法連絡您以通知有關 Azure RemoteApp 淘汰的訊息。 

我們已嘗試連絡您和其他 Azure 訂用帳戶管理員，以確認您了解 Azure RemoteApp 即將於 2017 年 8 月 31 日淘汰，我們[已在 2016 年 8 月 12 日宣布 Azure RemoteApp 的淘汰](http://aka.ms/araretirement) \(英文\)。   我們急需您回覆至 [arainfo@microsoft.com](mailto:arainfo@microsoft.com)，以表示：
1.  您和您的組織已了解將於 2017 年 8 月 31 日實施的淘汰措施
2.  告訴我們您移轉計畫的相關資訊 (如有需要，我們可提供協助)。 移轉選項已在[移出 Azure RemoteApp 的選項](http://aka.ms/aramigration)中詳細說明，選項包括 Citrix XenApp Essentials、MyCloudIT、Frame 等。 

Azure 管理入口網站中的 REMOTEAPP 節點可能已被封鎖，我們會這麼做是因為無法連絡上您，而必須強制您與我們連絡。  您有可能是因為與 Azure 訂用帳戶關聯的電子郵件地址未受監視或無效，而沒有收到我們的電子郵件。  我們強烈建議您確認與 Azure 訂用帳戶關聯的電子郵件地址為有效且受到監視，若要這麼做，請瀏覽至 Azure 入口網站並確認訂用帳戶系統管理員。  

仍有疑問或需要移轉上的支援？  請立即傳送電子郵件至 [arainfo@microsoft.com](mailto:arainfo@microsoft.com) 來連絡我們，或與我們的轉換小組[預約會議](http://aka.ms/ericorman) \(英文\)。
