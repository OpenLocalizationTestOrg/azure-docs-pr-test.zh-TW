# <a name="what-is-microsoft-genomics"></a>什麼是 Microsoft Genomics？
Microsoft Genomics 提供的雲端實作窩 Wheeler Aligner （哇哈哈哈哈） 以及次要的分析的 Genome 分析工具組 (GATK)。 服務是 ISO 認證，而且符合 HIPAA 法規，並提供價格 genome 排序需求的可預測性。 了解如何使用 Microsoft Genomics 服務，並與我們的 API 整合閱讀我們的快速入門中，教學課程中，與文件。

## <a name="support-your-most-demanding-sequencing-needs"></a>支援您最嚴格的定序需求
利用 Microsoft 的規模和經驗來執行 EB 規模的工作負載，而不用管理您自己的資料中心。 哇哈哈哈哈 GATK 的我們雲端倉儲實作的專屬是高 concordant 具有廣泛 Institute 的最佳作法的管線。


## <a name="keep-your-business-running"></a>讓公司運作
Microsoft Genomics 提供 99.99%可用性服務等級協定 (SLA) 來接收工作流程的要求。 深入了解 Microsoft Genomics [SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/)。


## <a name="secure-your-data"></a>保護您的資料
ISO 27001、 ISO 27018 和 ISO 9001 經過認證且相容的 HIPAA 法規 Microsoft Genomics 服務。 如需 Azure 平台安全性的相關資訊，請造訪 [Azure 信任中心](https://www.microsoft.com/TrustCenter/Security/default.aspx) \(英文\)。
