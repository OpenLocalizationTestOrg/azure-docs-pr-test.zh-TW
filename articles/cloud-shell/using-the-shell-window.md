# <a name="using-hello-azure-cloud-shell-window"></a>使用 hello Azure 雲端殼層 視窗

本文件說明如何 toouse hello 雲端殼層 視窗。

## <a name="concurrent-sessions"></a>並行工作階段
雲端殼層跨瀏覽器索引標籤可讓多個並行工作階段，藉由使用個別的 Bash 程序的每個工作階段 tooexist。
如果結束工作階段，請務必從每個工作階段視窗 tooexit 雖然它們在上執行每個處理序執行獨立 hello 相同的電腦。

## <a name="restart-cloud-shell"></a>重新啟動 Cloud Shell
![](media/recycle.png)
> [!WARNING]
> 重新啟動 Cloud Shell 將會重設電腦狀態，而所有未由檔案共用保存的檔案都將遺失。

* 按一下新的雲端 Shell 環境的 hello 工具列 tooreceive hello 重新啟動圖示。

## <a name="minimize--maximize-cloud-shell-window"></a>將 Cloud Shell 視窗最大化和最小化
![](media/minmax.png)
* 按一下 hello hello 圖示最小化右上角的 hello 視窗 toohide 它。 再按一下 hello 雲端殼層圖示 toounhide。
* 按一下 hello 最大化圖示 tooset 視窗 toomax 高度。 toorestore 視窗 tooprevious 大小，請按一下 [還原]。

## <a name="copy-and-paste"></a>複製和貼上
* Windows: `Ctrl-insert` toocopy 和`Shift-insert`toopaste。 在下拉式清單上按一下滑鼠右鍵，也能進行複製/貼上。
  * FireFox/IE 可能無法正確支援剪貼簿權限。
* Mac OS: `Cmd-c` toocopy 和`Cmd-v`toopaste。 在下拉式清單上按一下滑鼠右鍵，也能進行複製/貼上。

## <a name="resize-cloud-shell-window"></a>調整 Cloud Shell 視窗大小
* 按一下並拖曳 hello hello 工具列上的邊緣向上或向下 tooresize hello 雲端殼層 視窗。

## <a name="scrolling-text-display"></a>捲動文字顯示
* 使用滑鼠或觸控板 toomove 終端機文字捲動。

## <a name="exit-command"></a>結束命令
執行`exit`終止 hello 作用中工作階段。 此行為預設會在 20 分鐘後發生，不須進行互動。
