# <a name="storsimple-8000-series-update-22-release-notes"></a>StorSimple 8000 系列 Update 2.2 版本資訊
## <a name="overview"></a>概觀
下列版本資訊說明 StorSimple 8000 系列 Update 2.2 的新功能，並識別未決的重要問題。 當中也包含此版本中隨附之 StorSimple 軟體更新的清單。 

Update 2.2 可以套用至任何執行 Release (GA) 或 Update 0.1 到 Update 2.1 的 StorSimple 裝置。 與 Update 2.2 關聯的裝置版本為 6.3.9600.17708。

在 StorSimple 方案中部署更新之前，請檢閱版本資訊中所包含的資訊。

> [!IMPORTANT]
> * Update 2.2 僅提供軟體更新。 安裝此更新，大約需要花費 1.5-2 小時。 
> * 如果您執行的是 Update 2.1，建議您儘快套用 Update 2.2。
> * 如果是新版本，您可能不會立即看到更新，因為我們會分階段推出更新。 請等候數天然後再次掃描更新，因為很快就會提供更新。
> 
> 

## <a name="whats-new-in-update-22"></a>Update 2.2 的新功能
Update 2.2 包含以下重要的改良功能。

* **自動空間回收最佳化** – 在精簡佈建的磁碟區上刪除資料時，需回收未使用的儲存區塊。 此版本已經改善雲端空間回收程序，相較於之前的版本，此版本可讓未使用的空間更快變成可用空間。
* **快照效能增強** – Update 2.2 已經改進在某些情況 (使用大型磁碟區且資料變換的程度為最低到無的情況) 下處理雲端快照的時間。 可因這項增強功能獲益的案例是封存磁碟區。
* **強化支援封裝收集** – 此版本已改善收集及上傳支援封裝的方式。 
* **更新可靠性改進** – 此版本包含一些錯誤修正程式，可改進更新可靠性。

## <a name="controller-and-firmware-updates-in-update-22"></a>Update 2.2 中的控制器和韌體更新
此版本僅提供軟體更新。 但是，如果您是從 Update 2 之前的版本更新，您將需要在您的裝置上安裝驅動程式、Storport、Spaceport 及 (在某些情況下) 磁碟韌體更新。

如需有關如何安裝驅動程式、Storport、Spaceport 及磁碟韌體更新的詳細資訊，請參閱在您的 StorSimple 裝置上 [安裝 Update 2.2](storsimple-install-update-21.md) 。

## <a name="virtual-device-updates-in-update-22"></a>Update 2.2 中的虛擬裝置更新
這項更新無法套用至虛擬裝置。 將需要建立新的虛擬裝置。
