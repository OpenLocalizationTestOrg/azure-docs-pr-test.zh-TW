## <a name="users-and-pacing"></a><span data-ttu-id="8e240-195">使用者與步調</span><span class="sxs-lookup"><span data-stu-id="8e240-195">Users and pacing</span></span>
<span data-ttu-id="8e240-196">基準測試工作負載是由一個工具觸發，它會跨一組連接提交交易，以模擬許多並行使用者的行為。</span><span class="sxs-lookup"><span data-stu-id="8e240-196">The benchmark workload is driven from a tool that submits transactions across a set of connections to simulate the behavior of a number of concurrent users.</span></span> <span data-ttu-id="8e240-197">雖然所有的連接和交易都是由電腦產生，為了簡單起見，我們將這些連接視為「使用者」。</span><span class="sxs-lookup"><span data-stu-id="8e240-197">Although all of the connections and transactions are machine generated, for simplicity we refer to these connections as “users.”</span></span> <span data-ttu-id="8e240-198">雖然每一位使用者都獨立於其他所有使用者操作，但是所有使用者都執行相同的步驟循環，如下所示：</span><span class="sxs-lookup"><span data-stu-id="8e240-198">Although each user operates independently of all other users, all users perform the same cycle of steps shown below:</span></span>
