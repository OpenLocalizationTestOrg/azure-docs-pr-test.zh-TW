# <a name="problems-signing-in-tooan-custom-developed-application"></a>登入 tooan 自訂開發的應用程式的問題

有數個錯誤，導致您 toonot 無法 toosign 到應用程式。 hello 人員遇到此問題最大的原因是設定不正確的應用程式。

## <a name="errors-related-too-misconfigured-apps"></a>錯誤相關太設定不正確的應用程式

* 請確認您有應用程式中的 hello 入口網站中的這兩種 hello 設定相符。 特別是，比較用戶端/應用程識別碼、回覆 URL、用戶端密碼/金鑰，及應用程式識別碼 URI。

* 比較您正在要求存取 tooin 程式碼，以在 hello hello 設定權限的 hello 資源**所需的資源**您已設定 索引標籤 toomake 確定您只有要求的資源。

* 請參閱 [Azure AD StackOverflow](http://stackoverflow.com/questions/tagged/azure-active-directory) 中任何類似的錯誤或問題。
