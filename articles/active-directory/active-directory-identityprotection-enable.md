# <a name="enabling-azure-active-directory-identity-protection"></a>啟用 Azure Active Directory Identity Protection
Azure Active Directory Identity Protection 是一項新功能，可針對可疑的登入活動和潛在弱點提供整合檢視，並提供通知、補救建議及以風險為基礎的原則，來協助保護您的企業。 

hello 服務偵測到可疑活動的使用者和權限 （系統管理員） 身分識別基礎訊號像暴力密碼破解強制攻擊外, 洩認證，登入不熟悉的位置，從受感染的裝置，tooprotect 針對即時這些活動。 更重要的是，根據這些可疑的活動，使用者風險嚴重性計算並風險為基礎的原則可以設定自動保護您組織的 hello 身分識別。 如需詳細資料，請參閱 [Azure Active Directory Identity Protection](active-directory-identityprotection.md)。

本主題顯示如何 tooenable Azure Active Directory 識別身分保護。

## <a name="steps-tooenable-azure-active-directory-identity-protection"></a>步驟 tooenable Azure Active Directory 識別身分保護
1. [登入](https://ms.portal.azure.com/)tooyour 全域系統管理員身分的 Azure 入口網站。 
2. 在 hello Azure 入口網站，按一下  **Marketplace**。
   
    ![建立](./media/active-directory-identityprotection-enable/01.png "建立")
3. 在 hello 應用程式清單中，按一下 **安全性 + 身分識別**。
   
    ![建立](./media/active-directory-identityprotection-enable/02.png "建立")
4. 按一下 [Azure AD Identity Protection] 。
   
    ![建立](./media/active-directory-identityprotection-enable/03.png "建立")
5. 在 hello **Azure AD Identity Protection**刀鋒視窗中，按一下 **建立**。
   
    ![建立](./media/active-directory-identityprotection-enable/04.png "建立")
