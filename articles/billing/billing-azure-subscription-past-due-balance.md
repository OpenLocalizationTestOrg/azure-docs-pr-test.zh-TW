# <a name="resolve-past-due-balance-for-your-azure-subscription"></a>解決 Azure 訂用帳戶的逾期未付帳款 
如果未收到您的付款或我們無法處理您的付款，您可能會收到一封電子郵件，或在 [Azure 帳戶中心](https://account.windowsazure.com)或 [Azure 入口網站](https://portal.azure.com)中看到警示。 如果您是帳戶管理員且未收到以電子郵件寄送的帳單通知，[請檢查 Azure 帳戶系統管理員設定檔中的電子郵件地址](billing-how-to-change-azure-account-profile.md)。 建議您使用您會定期收信的電子郵件地址。

如果我們無法處理您的付款，您可能會收到一封含有如下訊息的電子郵件：

**我們已無法針對您的訂用帳戶向您的信用卡收取費用。若要避免服務發生中斷，請更新您的付款資訊。**

## <a name="if-you-forget-to-pay-what-happens"></a>如果您忘記付款，會發生什麼情況？
服務遭到取消，您的資源便無法使用。 服務終止 90 天後，就會刪除您的 Azure 資料。

## <a name="how-to-resolve-the-issue"></a>如何解決此問題
全額支付您未付的差額。

* 如果您使用發票付款模式，將您的付款寄至您的發票底部所列的地址。 如需協助，請連絡 [Azure 支援服務](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。

* 如果檔案中登記的信用卡遭拒或已過期，[請變更用於支付訂用帳戶的信用卡](billing-how-to-change-credit-card.md)，或連絡您的銀行以解決此問題。 當您更新付款方法時，Azure 會立即處理所有未付費用的付款。 您有未付費用之任何其他 Microsoft 服務的付款也會一併處理。

* 如果您因為帳戶管理員已離職或變更角色而沒有看到帳單，請連絡 [Azure 支援服務](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以解決此問題。

如果您知道您的付款已處理，但您的訂用帳戶仍然停用，請參閱[重新啟用已停用的 Azure 訂用帳戶](billing-subscription-become-disable.md)。

## <a name="need-help-contact-support"></a>需要協助嗎？ 請連絡支援人員。
如果仍需要協助，請[連絡支援人員](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以快速解決您的問題。
