# <a name="resolve-past-due-balance-for-your-azure-subscription-after-getting-an-email-from-azure"></a>收到 Azure 寄來的電子郵件之後，處理 Azure 訂用帳戶的逾期未付帳款

如果未收到您的付款或無法處理您的款項，您可能會收到電子郵件，或是在帳戶中心或 Azure 入口網站中看到通知。 該訊息可能如下所示：

**我們已無法針對您的訂用帳戶向您的信用卡收取費用。若要避免服務發生中斷，請更新您的付款資訊。**

請遵循本文的步驟以處理逾期未付帳款。

## <a name="resolve-the-past-due-balance"></a>處理逾期未付帳款

* 如果檔案中登記的信用卡遭拒或已過期，[請變更用於支付訂用帳戶的信用卡](billing-how-to-change-credit-card.md)，或連絡您的銀行以解決此問題。 當您更新付款方法時，Azure 會立即處理所有未付費用的付款。 您有未付費用之任何其他 Microsoft 服務的付款也會一併處理。

* 如果您使用發票付款模式，將您的付款寄至您的發票底部所列的地址。 如需協助，請連絡 [Azure 支援服務](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。

* 如果您因為帳戶管理員已離職或變更角色而沒有看到帳單，請連絡 [Azure 支援服務](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以解決此問題。

* 如果您已處理款項，但您的訂用帳戶狀態仍為停用，請連絡 [Azure 支援服務](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。

## <a name="not-getting-billing-email-notifications"></a>未收到帳單電子郵件通知嗎？

如果您是帳戶管理員，請[確認用於接收通知的電子郵件地址](billing-how-to-change-azure-account-profile.md)。 建議您使用您會定期收信的電子郵件地址。 如果電子郵件正確，請查看垃圾郵件資料夾。

## <a name="if-i-forget-to-pay-what-happens"></a>如果忘記付款會如何？

服務遭到取消，您的資源便無法使用。 服務終止 90 天後，就會刪除您的 Azure 資料。 若要深入了解，請參閱 [Microsoft 信任中心 - 我們如何管理您的資料](https://go.microsoft.com/fwLink/p/?LinkID=822930&clcid=0x409) \(英文\)。

## <a name="need-help-contact-support"></a>需要協助嗎？ 請連絡支援人員。

如果仍需要協助，請[連絡支援人員](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以快速解決您的問題。
