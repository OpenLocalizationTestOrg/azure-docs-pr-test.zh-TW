### <a name="how-it-works"></a><span data-ttu-id="b7802-361">運作方式</span><span class="sxs-lookup"><span data-stu-id="b7802-361">How it works</span></span>

<span data-ttu-id="b7802-362">當使用者在登入時勾選「**X** 天內不要再問我」的方塊時，記住 Multi-Factor Authentication 的運作方式是在瀏覽器上設定永續性 cookie。</span><span class="sxs-lookup"><span data-stu-id="b7802-362">Remembering Multi-Factor Authentication works by setting a persistent cookie on the browser when a user checks the "Don't ask again for **X** days" box at sign-in.</span></span> <span data-ttu-id="b7802-363">系統不會再次提示使用者從該瀏覽器輸入 MFA，直到 cookie 到期為止。</span><span class="sxs-lookup"><span data-stu-id="b7802-363">The user won't be prompted for MFA again from that broswer until the cookie expires.</span></span> <span data-ttu-id="b7802-364">如果使用者在相同的裝置上開啟不同的瀏覽器或清除其 cookie，系統會提示他們再次確認。</span><span class="sxs-lookup"><span data-stu-id="b7802-364">If the user opens a different browser on the same device or clears their cookies, they are prompted to verify again.</span></span>
