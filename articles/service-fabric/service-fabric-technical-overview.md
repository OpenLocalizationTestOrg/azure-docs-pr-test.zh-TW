<span data-ttu-id="7bdc6-122">**應用程式封裝**：磁碟目錄，包含此應用程式類型的 `ApplicationManifest.xml` 檔案。</span><span class="sxs-lookup"><span data-stu-id="7bdc6-122">**Application Package**: A disk directory containing the application type's `ApplicationManifest.xml` file.</span></span> <span data-ttu-id="7bdc6-123">參考組成此應用程式類型的每個服務類型的服務封裝。</span><span class="sxs-lookup"><span data-stu-id="7bdc6-123">References the service packages for each service type that makes up the application type.</span></span> <span data-ttu-id="7bdc6-124">應用程式封裝目錄中的檔案會複製到 Service Fabric 叢集的映像存放區。</span><span class="sxs-lookup"><span data-stu-id="7bdc6-124">The files in the application package directory are copied to Service Fabric cluster's image store.</span></span> <span data-ttu-id="7bdc6-125">例如，電子郵件應用程式類型的應用程式封裝可能包含指向佇列服務封裝、前端服務封裝、資料庫服務封裝的參考。</span><span class="sxs-lookup"><span data-stu-id="7bdc6-125">For example, an application package for an email application type could contain references to a queue service package, a frontend service package, and a database service package.</span></span>

<span data-ttu-id="7bdc6-136">服務分為兩種：</span><span class="sxs-lookup"><span data-stu-id="7bdc6-136">There are two types of services:</span></span>
