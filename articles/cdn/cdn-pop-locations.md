# <a name="azure-cdn-pop-locations"></a>Azure CDN 的 POP 位置

> [!IMPORTANT]
> \***來自 Akamai 的 Azure CDN** POP 位置不會個別揭露。  
> 
> 這兩個提供者可以不同方式建置其 CDN 基礎結構。  建議您不要使用 POP 位置來決定要使用哪個 Azure CDN 產品，並考慮改用功能和使用者效能。  測試兩個提供者的效能來為使用者選擇正確的 Azure CDN 產品。
