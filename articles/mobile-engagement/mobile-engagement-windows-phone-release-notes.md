# <a name="windows-phone-silverlight-sdk-release-notes"></a>Windows Phone Silverlight SDK 版本資訊
##<a name="331-11032016"></a>3.3.1 (11/03/2016)
*MicrosoftAzure.MobileEngagement* Nuget 封裝 **v3.4.1**

* 穩定性改進。

## <a name="330-04192016"></a>3.3.0 (04/19/2016)
*MicrosoftAzure.MobileEngagement* Nuget 封裝 **v3.4.0**

* 已加入 "TestLogLevel" API 來啟用/停用/篩選 SDK 所發出的主控台記錄檔。

## <a name="320-01222016"></a>3.2.0 (01/22/2016)
*MicrosoftAzure.MobileEngagement* Nuget 封裝 **v3.3.0**

* 動作 URL 現在可使用自訂配置、ms-appx 及 ms-appdata。
* 穩定性改進。

## <a name="310-11202015"></a>3.1.0 (11/20/2015)
*MicrosoftAzure.MobileEngagement* Nuget 封裝 **v3.2.0**

* 修正初始化之後立即要求裝置 ID 時發生的當機。
* 主控台記錄檔改善。

## <a name="300-04032015"></a>3.0.0 (2015/04/03)
* 預設通知圖示已更新。
* 按一下通知時，會傳回系統通知動作回饋。
* 已修正系統通知的競爭條件。

## <a name="200-02172015"></a>2.0.0 (2015/02/17)
* Azure Mobile Engagement 的最初發行版本。
* 以連接字串組態取代 appId/sdkKey 組態。
* 增強安全性。
