## <a name="introduction"></a>簡介
Azure IoT 中樞是一項完全受管理的服務，可在數百萬個物聯網 (IoT) 裝置和一個解決方案後端之間啟用可靠且安全的雙向通訊。 其中一個 hello IoT 專案面對的挑戰是如何 tooreliably 安全地連線裝置 toohello 方案後端。 tooaddress 這項挑戰，IoT 中樞：

* 提供可靠的裝置到雲端和雲端到裝置的超大規模傳訊。
* 使用每一裝置的安全性認證和存取控制來啟用安全通訊。
* 包含裝置 hello 最受歡迎的語言與平台的程式庫。

本教學課程說明如何：

* 使用 hello Azure 入口網站 toocreate IoT 中樞。
* 在您的 IoT 中樞中建立裝置身分識別。
* 建立模擬的裝置應用程式傳送遙測 tooyour 方案後端。
