Azure Data Factory 是資料整合服務。 可讓您在雲端中建立資料驅動的工作流程。 工作流程會以一個或多個「管線」進行實作。 管線會協調及自動處理資料動向及資料轉換。 管線可以為您的資料執行下列程序：

1. 從不同資料存放區內嵌資料。
2. 使用以下計算服務來轉換或處理資料：
    - Azure HDInsight Hadoop
    - Spark
    - Azure Data Lake Analytics
    - Azure Machine Learning
3. 將輸出資料發行至資料存放區。
    - 發行目標可以是 Azure SQL 資料倉儲，以便讓商業智慧 (BI) 應用程式取用。 

您可以使用 Data Factory 排程管線。
