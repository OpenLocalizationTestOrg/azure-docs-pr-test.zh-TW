
## <a name="launch-azure-cloud-shell"></a>啟動 Azure Cloud Shell

Azure Cloud Shell 是免費的互動式 Shell，可讓您用來執行本文中的步驟。 它具有預先安裝和設定的共用 Azure 工具，可與您的帳戶搭配使用。 只要按一下 [複製] 即可複製程式碼，將它貼到 Cloud Shell 中，然後按 enter 鍵加以執行。 以下有幾種啟動 Cloud Shell 的方式：

|  |   |
|-----------------------------------------------|---|
| 按一下程式碼區塊右上角的 [試試看]。 | ![本文中的 Cloud Shell](./media/cloud-shell-powershell/cloud-shell-powershell-try-it.png) |
| 在您的瀏覽器中開啟 Cloud Shell。 | [![https://shell.azure.com/powershell](./media/cloud-shell-powershell/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 按一下 Azure 入口網站右上方功能表上的 [Cloud Shell] 按鈕。 | [![入口網站中的 Cloud Shell](./media/cloud-shell-try-it/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |
