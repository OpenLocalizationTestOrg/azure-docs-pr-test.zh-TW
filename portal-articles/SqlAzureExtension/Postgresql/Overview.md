# <a name="azure-database-for-postgresql-documentation"></a>適用於 PostgreSQL 的 Azure 資料庫文件

Azure PostgreSQL 資料庫是根據 hello 開放原始碼 Postgres 資料庫引擎的關聯式資料庫服務。 它是一個完全受管的資料庫即服務，可透過可預測效能、安全性、高可用性及動態延展性處理任務關鍵性工作負載。  開發 Azure Database PostgreSQL 運用 hello 開放原始碼工具和您選擇的平台的應用程式。  深入了解如何 toouse Azure 資料庫，以獲得 PostgreSQL 與我們的快速入門中，教學課程和範例。
