# <a name="azure-database-for-postgresql-documentation"></a>適用於 PostgreSQL 的 Azure 資料庫文件

適用於 PostgreSQL 的 Azure 資料庫是以開放原始碼 Postgres 伺服器引擎為基礎的關聯式資料庫服務。 它是一個完全受管的資料庫即服務，可透過可預測效能、安全性、高可用性及動態延展性處理任務關鍵性工作負載。  您可以使用開放原始碼工具，以及您選擇的平台，透過適用於 PostgreSQL 的 Azure 資料庫來開發應用程式。  了解如何透過我們的快速入門、教學課程和範例使用適用於 PostgreSQL 的 Azure 資料庫。
