- [建立 Windows 虛擬機器](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot)
- [建立 Linux 虛擬機器](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot)
